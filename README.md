# Real-World Robot Systems Portfolio

This page showcases selected projects involving **real-world robot deployment**, spanning mobile robots, mobile manipulators, quadruped, and custom-built humanoid systems.

Across these platforms, I have worked on:
- Autonomy and reactive control
- Vision-language integration
- Loco-manipulation
- Hardware debugging and system integration
- Deployment on physical robotic systems (ROS-based stacks)

Each project page contains further technical details.
